# 마윤자 — GLP-1 트래커

위고비(Wegovy, 세마글루타이드) / 마운자로(Mounjaro, 티르제파타이드) 같은 GLP-1 비만 치료제를
스스로 관리하기 위한 단일 사용자용 웹앱. **마윤자**(마운자로 + 위고비) 디자인을 적용했습니다.

## 화면

- **홈** — 다음 주사 D-day 큰 카드, 현재 용량, 체중 요약, 오늘의 컨디션, 최근 부작용, 빠른 액션
- **주사 일정** — 다가오는 주사, 월간 캘린더(주사일/예정일 마커), 부위 로테이션 추천, 주사 로그
- **체중 기록** — 시작/현재/감량/목표 진행률, 최근 12회 막대 차트, 허리·체지방·기록일수, 오늘 입력
- **용량 단계** — 표준 4주 티트레이션 타임라인(완료/진행중/예정), 의사 처방 정보
- **부작용 기록** — 횟수/평균 강도/기록일수 통계, 자주 발생한 부작용 막대, 인사이트, 강도 1~10 슬라이더

## 기술 스택

- Next.js 16 (App Router) + TypeScript + Turbopack
- Tailwind CSS v4 + **shadcn/ui (base-ui)**
- Recharts (체중 막대 차트, 인사이트)
- date-fns, react-day-picker
- 폰트: **Geist + JetBrains Mono** (숫자/단위는 모노)
- 데이터는 **브라우저 localStorage** (서버 없음, 단일 디바이스)

## 디자인 토큰

마윤자 Pencil 디자인 파일(`/Users/maru/dev/maja.pen`, lunaris 시스템 기반)에서 추출:

- 페이지 배경 `#F2F3F0`, 카드 `#FFFFFF`, 텍스트 `#111111`, 보조 텍스트 `#666666`
- 강조색 **`#2196F3`** (시안의 D-day 카드 / 브랜드 버튼) — `--brand` / `--brand-soft`
- 다크모드는 OS 시스템 설정 기반 자동 전환
- 라디우스 16px, 알약 999px

## 실행

```bash
pnpm install   # 최초 1회
pnpm dev       # http://localhost:3000
pnpm build     # 프로덕션 빌드
```

## 데이터 키

브라우저 DevTools → Application → Local Storage 에서 확인:

- `glp1.profile` — 사용자 이름·약·시작일
- `glp1.injections` — 주사 기록
- `glp1.weights` — 체중 기록
- `glp1.notes` — 부작용 기록
- `glp1.doseStages` — (예약) 사용자 정의 단계

## 의도적으로 하지 않는 것

- 로그인/멀티 디바이스 동기화/서버
- 사진 업로드
- AI 자동 인사이트(현재는 단순 카운트)
- 식단 페이지(디자인에 없음 — 옛 `/meals` 라우트는 잠시 남겨둠)
